What is Cosy?

Cosy is a set of JavaScript tools and libraries inspired by the LISP and Clojure programming languages. It aims to help us become lazier developers that write lazier code.

So far only the cosy-lang library exists which adds many of the language features that the clojure guys enjoy including:

  1. Protocols
  2. Sequences (lazy & asyncronous)
  3. Argument length dispatch
  4. Tail recursion
  5. Object Identity
  6. Object Metadata

It's still early days yet, but feel free to contribute in any way you can.
